# 让 div 水平垂直居中

方法 1：

```css
.parent {
  display: flex; /* 换成 display:grid; 也可 */
}
.child {
  margin: auto;
}
```

方法 2：

```css
.parent {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

方法 3：

```css
.parent {
  position: relative;
}
.child {
  position: absolute;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
}
```

方法 4：

```css
.parent {
  display: table-cell;
  text-align: center;
  vertical-align: middle;
}

.child {
  display: inline-block;
}
```

方法 5：

```css
.parent {
  position: relative;
}

.child {
  margin: auto;
  position: absolute;
  top: 0;
  left: 0;
  bottom: 0;
  right: 0;
}
```
